# 스프링 시큐리티 연습

## 프로젝트 구성
java 8+  
spring boot 2.4+  
h2 database  
gradle 6.5+
## 챕터별 설명
 - 챕터 0 - 시큐리티 맛보기
    - 폼기반 인증, 암호화 처리, 인가처리, rememeber-me처리등 시큐리티 전반에 대한 기능 맛보기 프로젝트  
 - 챕터 1 - 시큐리티 프로젝트 구성
 - 챕터 2 - 사용자 정의 보안 기능 구현
 - 챕터 3 - 폼(form) 로그인 인증
 - 챕터 4 - UsernamePasswordAuthenticationFilter의 이해
 - 챕터 5 - Logout(로그아웃), LogoutFilter
